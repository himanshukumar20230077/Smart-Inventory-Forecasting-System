# 📦 Smart Inventory Forecasting & Analytics System

## 🚀 Overview

An AI-powered inventory analytics platform built using:

- Python
- Machine Learning
- Streamlit
- SQL
- Power BI

This system allows businesses to:

- Upload inventory CSV files
- Generate analytics dashboards
- Forecast future sales demand
- Analyze product performance
- Download processed reports

---

## 🔥 Features

✅ Dynamic CSV Upload
